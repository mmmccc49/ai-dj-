# AI DJ 产品介绍

AI DJ 是一个面向个人使用的桌面电台播放器。它把网易云音乐播放、AI 导播、语音播报和歌单搜索放在同一个小窗口里，适合日常工作、开车前选歌、夜间听歌或临时生成某种情绪的歌单。

## 核心能力

- 网易云音乐播放：支持每日推荐、收藏歌单、心动模式、搜索歌曲和搜索歌单。
- AI 电台 DJ：接入 OpenAI-compatible 模型后，可以用自然语言点歌、找歌单、聊音乐和自动串词。
- MiMo 语音播报：AI 回复后可生成 DJ 语音，并在播报时自动降低背景音乐音量。
- 本地设置面板：安装后只需要在界面里填写网易云 Cookie、AI 接口 URL、模型名称和 Key。
- 桌面小窗模式：支持普通播放器窗口和迷你窗口切换。

## 适合谁使用

- 想把网易云音乐变成“私人电台”的用户。
- 想测试 OpenAI-compatible 模型在音乐导播场景中表现的开发者。
- 想用 MiMo 语音模型做实时语音播报体验的人。

## 使用流程

1. 下载并安装 Windows 或 macOS 安装包。
2. 打开 AI DJ。
3. 点击歌单标题右侧的齿轮按钮。
4. 填写网易云 Cookie、OpenAI-compatible DJ 接口、MiMo 语音接口。
5. 保存后回到播放器，点击每日推荐、收藏歌单或直接和 DJ 对话。

## 配置要求

AI DJ 不内置任何第三方账号密钥。每个用户需要准备自己的配置：

- 网易云 Cookie：用于读取账号相关音乐内容。
- OpenAI-compatible 接口：用于 AI DJ 文本回复和动作决策。
- MiMo 语音接口：用于把 DJ 串词变成语音。

所有配置保存在用户本机，不应上传到 GitHub 或公开分享。

## 发布形式

项目使用 Electron 打包：

- Windows：NSIS 安装包。
- macOS：DMG 和 ZIP。

GitHub Actions 会在代码推送或创建版本 tag 时自动构建安装包。创建 `v*` tag 时，安装包会附加到 GitHub Release，方便用户直接下载。
