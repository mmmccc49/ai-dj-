# AI DJ 项目介绍

AI DJ 的目标很简单：让普通用户也能拥有一个会说话、会找歌、会串场的私人音乐电台。

传统播放器只负责播放音乐。AI DJ 在播放器之外加了一层“电台导播”：它可以理解你的自然语言请求，知道当前正在播放什么歌，也能根据你的心情或场景去网易云音乐里找歌、找歌单，并用电台 DJ 的方式做简短介绍。

## 核心卖点

### 只需要填写对应 API

用户不需要改代码、不需要懂 Electron、不需要自己部署服务。安装软件后，在设置面板里填写：

- 网易云 Cookie
- OpenAI-compatible 接口 URL、模型名称、Key
- MiMo 语音接口 URL、模型名称、Key

保存后立即生效。配置会保存在本机用户目录中，下一次打开自动读取。

### 把“找歌”变成对话

用户可以像和电台主持人聊天一样说：

- “来点适合开车的。”
- “想听王菲，安静一点的。”
- “现在这首歌帮我介绍一下。”
- “放点晚上工作不吵的歌。”

AI DJ 会把这些话转换成歌曲、歌手或歌单搜索，并自动播放最匹配的结果。

### 自动生成电台串词

每首歌开始播放时，AI DJ 可以根据歌曲、歌手、专辑、上一首歌和当前时间生成短串词。它不是单纯报歌名，而是更像真实电台里的过渡：说一点氛围、听感、故事或场景。

### 语音播报和自动压低音乐

接入 MiMo 语音模型后，DJ 串词可以直接生成语音。语音播放时，背景音乐会自动降低音量；播报结束后，音乐音量再恢复。

### 安装包直接分发

项目通过 GitHub Actions 自动打包：

- Windows：NSIS 安装包
- macOS：zip 应用包

创建版本 tag 后，安装包会自动上传到 GitHub Release，用户只需要下载对应系统的安装包。

## 适合的用户

- 想把网易云音乐变成私人电台的用户。
- 想用自己的 AI API 做音乐导播体验的人。
- 想测试 OpenAI-compatible 模型在音乐场景中表现的开发者。
- 想把 MiMo 语音模型用于实时播报体验的人。

## 使用门槛

AI DJ 不内置任何第三方密钥。每个用户准备好自己的账号 Cookie 和 API Key 后，就可以在图形界面里完成配置。整个过程不需要命令行，不需要改配置文件。

这也是它最重要的便利性：下载、安装、打开设置、填 API、保存，然后就能听。
