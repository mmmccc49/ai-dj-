# Cloudio Player Figma Import

This folder contains an editable SVG mockup for Figma:

- `cloudio-player-figma-import.svg`

## How to Use

1. Open Figma.
2. Drag `cloudio-player-figma-import.svg` onto the canvas, or use `File > Place image/video`.
3. Right-click the imported SVG and ungroup as needed.
4. Main layers are grouped as:
   - `Titlebar`
   - `Player Area`
   - `Album Cover`
   - `Track Details`
   - `Progress`
   - `Transport Controls`
   - `Playlist Area`
   - `Track List`
   - `Status Bar`

## Design Tokens

- Background: `#0A0A0F`
- Surface: `#12121A`
- Raised surface: `#1A1A24`
- Text: `#F0ECE4`
- Muted text: `#F0ECE4` at 50% opacity
- Faint text: `#F0ECE4` at 30% opacity
- Accent: `#D4A853`
- Accent bright: `#E8BF6A`

## Frame

- Full player frame: `480 x 800`
- Header: `44px`
- Compact player area: `260px`
- Playlist area: fills the remaining space
- Status bar: `28px`
